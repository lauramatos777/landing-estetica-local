# Landing Estética Local

Projeto de portfólio para simular uma landing page profissional de um negócio fictício de estética e bem-estar.

## Estrutura inicial

- `index.html` - página principal.
- `styles/style.css` - estilos da interface.
- `scripts/main.js` - interações leves e animações.
- `assets/images/` - imagens futuras do projeto.
